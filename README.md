# 🎨 Agnes Image 2.1 Flash 生成器

基于 Agnes AI API 的图像生成工具，支持文生图、图生图、批量生成。

## 使用方法

1. 在界面中输入你的 Agnes API Key（从 [agnes-ai.com](https://agnes-ai.com) 获取）
2. 输入提示词生成图片
3. 支持多种尺寸，包括抖音适配尺寸
